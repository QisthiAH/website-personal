About Me

Halo, saya Qisthi Annisa mahasiswa undergraduate jurusan informatika Universita Siliwangi. Adapun beberapa pengalaman saya di bidang IT adalah pernah membuat aplikasi mobile “Tukang Anak”, yaitu aplikasi yang berkaitan dengan kesehatan anak, khususnya dalam menangani masalah stunting. Selain itu, saya juga memiliki pengalaman dalam pengembangan prototipe aplikasi untuk industri lainnya seperti prototipe aplikasi “MoCut”, yaitu aplikasi untuk memudahkan pengguna memanggil tukang cukur ke tempat yang diinginkan jika pengguna tidak memiliki waktu untuk mengunjungi tempat cukur.

My Skill

Sebagai seorang desainer yang fokus pada penggunaan Figma, saya memiliki kemampuan untuk menciptakan pengalaman desain yang fungsional bagi pengguna. Skill penggunaan figma ini didukung oleh skill programming flutter untuk front end dan firebase untuk backend sehingga desain yang telah dibuat bisa langsung diimplementasikan untuk pembuatan aplikasi mobile ataupun web yang responsif. Selain itu, saya juga mempunyai skill programming python yang mana hal ini biasa saya gunakan jika programming sangat berkaitan dengan hal yang bersifat data atu matematis.
